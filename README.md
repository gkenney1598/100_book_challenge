# 100 Best Books Reading Challenge
Read the Best Books of the 21st Century, according to the [NYT](https://www.nytimes.com/interactive/2024/books/best-books-21st-century.html)!

## Built Using
+React.js
+TypeScript
+HTML/CSS
+Vite
